This project simulates a load balancer that distributes network requests to web servers. The system handles requests by assigning them to available servers, each of which processes the request and tracks its status. The simulation allows for the dynamic addition and removal of servers based on the request load.

The main components of the simulation include\+:


\begin{DoxyItemize}
\item {\bfseries{\doxylink{struct_request}{Request}}}\+: Represents a network request with metadata such as source and destination IPs, processing time, and job type.
\item {\bfseries{\doxylink{class_request_queue}{Request\+Queue}}}\+: Manages a queue of requests to be processed by the load balancer.
\item {\bfseries{\doxylink{class_web_server}{Web\+Server}}}\+: Represents a server that processes assigned requests.
\item {\bfseries{\doxylink{class_load_balancer}{Load\+Balancer}}}\+: Manages the distribution of requests to web servers.
